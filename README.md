# CSC-450-Group-Project
## Requirements
- Windows with Python 3 and Git on PATH, or
- Linux
## Setup
### Windows
```
git clone https://github.com/lpitts1/CSC-450-Group-Project.git flashcards
python3 -m venv flashcards
flashcards\Scripts\pip install PyQt6
```
### Linux
```
git clone https://github.com/lpitts1/CSC-450-Group-Project.git flashcards
python3 -m venv flashcards
flashcards/bin/pip install PyQt6
```
## Use
- Windows: `\path\to\flashcards\Sscripts\python \path\to\flashcards\ProjectUI.py`
- Linux: `/path/to/flashcards/bin/python /path/to/flashcardsProjectUI.py`
